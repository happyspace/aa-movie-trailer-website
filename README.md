# Movie Trailer Website -- Project 

Movie trialer web page.

## How to launch

### From IDLE 
 + Choose 'File -> Open'
 + Navigate to 'entertainment_center.py' 
 + Choose 'Run -> Module'
 
### From CMD
 + Navigate to project directory 
 + Run for following command
 > py entertainment_center.py
